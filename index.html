📁 Data Storage Options

For SharePoint:

1. Export your data using the "Export Data" button

2. Save the JSON file to a SharePoint document library

3. Share the file link with your team members

4. Team members can download and import using "Import Data"

For Confluence:

1. Export your data and attach the JSON file to a Confluence page

2. Or create a Code Block macro with json language and paste the exported data

3. Team members can copy the JSON and save as a file to import

Auto-sync Option:

For automatic syncing, consider hosting the JSON file on a web server and modifying the code to fetch from a URL on load.
